# ⚡ KNUT OBFUSCATOR

Protégez votre code JavaScript avec la puissance de l'obfuscation moderne.

## ✨ Fonctionnalités

| Symbole | Fonctionnalité | Description |
|---------|---------------|-------------|
| ⌂ | **Accueil** | Page de présentation avec photo de profil |
| ◈ | **Obfuscation Simple** | Collez votre code et obfusquez-le instantanément |
| ◉ | **Obfuscation ZIP** | Chargez des projets complets au format ZIP |
| ⌗ | **Options avancées** | Compact, self-defending, string array, debug protection |
| ⚡ | **Performance** | Obfuscation rapide et optimisation du code |
| ⌘ | **Actions** | Copier, télécharger, voir l'original |

## 🛡️ Techniques d'obfuscation

- **Renommage de variables** : Noms hexadécimaux (_0x1234)
- **Encodage des strings** : Tableau de strings encodées
- **Code mort** : Injection de fonctions inutiles
- **Anti-debug** : Protection contre le débogage
- **Self-defending** : Anti-modification du code
- **Désactivation console** : Bloque les logs
